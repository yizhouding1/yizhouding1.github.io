# Yizhou Ding — Academic Website

A static academic website designed for GitHub Pages.

## Navigation structure

The site uses four content panels and one direct PDF link:

1. **Profile** — biography, education, languages, tests, and contact
2. **Research** — manuscript and Galaxy Futures empirical project
3. **Campus** — teaching, Nectar course project, Tri-Service, and community service
4. **Awards** — award list and selected visual documentation
5. **Vita** — opens the CV PDF directly

Only the selected panel is displayed. The panels are switched by `script.js` and can also be opened through URL hashes such as `#research`.

## Deploy to GitHub Pages

Upload all files and folders in this directory to the root of the GitHub repository. Keep these paths unchanged:

- `index.html`
- `styles.css`
- `script.js`
- `files/`
- `images/`

Then enable GitHub Pages from the repository's main branch and root directory.
